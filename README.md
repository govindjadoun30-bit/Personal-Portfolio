# Personal Portfolio Website

## Project Overview

This project is a responsive personal portfolio website created using HTML, CSS, and JavaScript.

The website introduces me as a B.Tech Computer Science Engineering student and Web Development Intern. It showcases my skills, personal information, and contact form in a clean and visually appealing design.

The main objective of this project is to practice and implement CSS concepts such as colors, gradients, Flexbox, hover effects, background images, transitions, and responsive design.

---

## Features

- Personal introduction section
- Profile photo
- About Me section
- Skills section
- Interactive hover effects on skill cards
- Contact form
- Form validation using HTML
- Message sent button interaction using JavaScript
- Gradient backgrounds

## Design Features

### Header Section

The header uses a multi-color linear gradient background and contains:

- Name
- Professional role
- Career goal
- Contact button

The Contact Me button includes a hover effect that changes its color and position.

### About Section

The About section includes:

- Profile photo
- Personal introduction
- Career goal
- Sky background image
- Decorative leaf elements

The profile image has curved corners while maintaining a square shape.

### Skills Section

The Skills section displays the following skills:

- HTML
- CSS
- Java
- JavaScript
- Git & GitHub

The skill cards use CSS Flexbox and include hover effects.

When the user moves the mouse over a skill card:

- The background color changes
- The text color changes
- The card moves slightly upward
- The card slightly increases in size

### Contact Section

The Contact section includes:

- Name input
- Email input
- Message textarea
- Send Message button

The section also includes a telephone icon as a decorative background element.

After submitting the form, the button changes to:

`Message Sent ✓`

and its background color changes to green using JavaScript.
